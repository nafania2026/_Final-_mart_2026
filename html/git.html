Git

  • git init - с помощью неё инициируется новый репозиторий.
  • git clone - с помощью неё происходит клонирование удалённого репозитория.
  • git add - добавляет содержимое рабочего каталога в индекс (stading area) для последующего коммита.
  • git status Проверка состояния репозитория.
  • git diff - Просмотр изменений.
